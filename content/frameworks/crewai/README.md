# CrewAI

## Summary

Popular framework for role-based agent collaboration and task decomposition.

## Best for

- role-driven agent teams
- workflow demos and prototypes
- nontrivial task decomposition

## Why it matters

CrewAI is widely used in demos and tutorials, so it is a helpful ecosystem reference even if you choose a different core framework.
