# Promptflow

## Summary

End-to-end framework from Microsoft for building, testing, and operating LLM applications.

## Best for

- prompt and flow prototyping
- operational workflows around LLM apps
- teams that want evaluation and deployment in one ecosystem

## Why it matters

Useful when you want a more workflow-centric approach than a pure agent SDK.
