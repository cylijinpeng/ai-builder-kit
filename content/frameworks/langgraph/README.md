# LangGraph

## Summary

Graph-based agent framework from the LangChain ecosystem, designed for controllable, stateful workflows.

## Best for

- graph-shaped workflows
- durable state and checkpoints
- complex orchestration with explicit control flow

## Why it matters

Strong option when you want more control over branching, persistence, and workflow state than a simple chat loop gives you.
