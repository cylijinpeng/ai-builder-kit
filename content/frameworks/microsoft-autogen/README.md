# Microsoft AutoGen

## Summary

Well-known multi-agent framework from Microsoft with support for agent chat, tools, MCP integration, and orchestration patterns.

## Best for

- multi-agent experiments
- agent teams and delegation patterns
- Python-heavy stacks

## Why it matters

AutoGen remains one of the most recognizable agent frameworks and is valuable as both a learning resource and a production reference.
