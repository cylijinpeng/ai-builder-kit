# OpenAI Agents SDK (JavaScript/TypeScript)

## Summary

Official JavaScript and TypeScript SDK for agent workflows, tools, guardrails, tracing, and voice agents.

## Best for

- Node.js agent systems
- browser or server-side JavaScript stacks
- teams that want built-in tracing and handoffs

## Why it matters

Useful when you want an official JS/TS agent stack instead of wiring orchestration pieces together yourself.
