# DSPy

## Summary

Framework from Stanford for programming and optimizing language model pipelines rather than hand-tuning prompts alone.

## Best for

- prompt optimization workflows
- research-oriented LLM programming
- teams exploring evaluation-driven iteration

## Why it matters

DSPy is one of the strongest references if you want to move from manual prompting toward more systematic optimization.
