# OpenAI Agents SDK (Python)

## Summary

Official Python SDK for building multi-agent workflows with tools, handoffs, guardrails, and tracing.

## Best for

- Python-based agent systems
- tool-using agents
- production-oriented orchestration patterns

## Why it matters

Strong default choice when you want an official Python agent framework with a relatively small conceptual surface area.
