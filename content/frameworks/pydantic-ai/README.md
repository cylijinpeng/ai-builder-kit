# PydanticAI

## Summary

Python agent framework that leans into strong typing, validation, and structured outputs.

## Best for

- type-safe agent systems
- Python applications using Pydantic already
- structured tool and response flows

## Why it matters

Useful for teams that care about typed interfaces and predictable structured outputs more than maximal framework abstraction.
