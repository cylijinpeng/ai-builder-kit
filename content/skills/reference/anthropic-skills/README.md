# Anthropic Skills

## Summary

Official public repository for Anthropic's skills ecosystem, with many real examples and a linked open standard.

## Best for

- understanding real skill packaging
- seeing advanced skill examples
- learning how skills combine instructions, scripts, and references

## Why it matters

One of the strongest public references for what high-quality, reusable skills look like in practice.
