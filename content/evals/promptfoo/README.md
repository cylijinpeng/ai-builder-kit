# promptfoo

## Summary

Practical testing and red-teaming tool for prompts, agents, and RAG systems.

## Best for

- CI checks for prompts
- quick side-by-side model comparisons
- security and prompt-injection testing

## Why it matters

Especially useful if you want evaluation workflows that are easier to operationalize than a research-style benchmark repo.
