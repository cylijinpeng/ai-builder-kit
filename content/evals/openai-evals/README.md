# OpenAI Evals

## Summary

Open-source framework and benchmark registry for evaluating LLM behavior and application quality.

## Best for

- custom eval creation
- benchmark references
- evaluation-first development loops

## Why it matters

Evals are one of the highest-leverage practices in real LLM development, and this repo is a strong reference for how to structure them.
