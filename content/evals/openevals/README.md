# OpenEvals

## Summary

LangChain ecosystem project for ready-to-use evaluators and evaluation helpers.

## Best for

- evaluation components you can plug into apps quickly
- teams already using LangChain or LangGraph
- smaller evaluation setups with reusable building blocks

## Why it matters

Useful middle ground between hand-rolled checks and full custom evaluation frameworks.
