# GitHub prompt files

## Summary

Official GitHub Models guidance for storing prompts directly in repositories as `.prompt.yml` or `.prompt.yaml` files.

## Best for

- versioning prompts in Git
- testing prompt changes
- sharing prompts with technical and non-technical collaborators

## Why it matters

Prompt files turn prompts into first-class repository assets. This repo uses the same format for some starter prompts.

- Docs: https://docs.github.com/en/github-models/use-github-models/storing-prompts-in-github-repositories
