# GitHub Copilot custom instructions

## Summary

Official GitHub documentation for repository-wide Copilot guidance using `.github/copilot-instructions.md`.

## Best for

- repository onboarding for Copilot
- code review behavior guidance
- documenting build and test expectations

## Why it matters

If your repository supports GitHub Copilot, this is the fastest official way to encode project-specific expectations without repeating them in every chat.

## Reference

- Docs: https://docs.github.com/en/copilot/how-tos/configure-custom-instructions/add-repository-instructions
