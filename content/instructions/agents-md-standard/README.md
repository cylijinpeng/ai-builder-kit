# AGENTS.md standard

## Summary

Open, repository-local instruction format for coding agents.

## Best for

- giving agents project context
- documenting coding conventions
- reducing repeated setup prompts

## Why it matters

AGENTS.md is becoming a common way to give coding agents a predictable place to look for repository-specific guidance. It is especially useful when you want Codex, Claude Code, Copilot, or other tools to behave consistently inside a project.
